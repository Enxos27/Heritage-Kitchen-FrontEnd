# 🥗 Heritage Kitchen - Frontend 

L'interfaccia utente di **Heritage Kitchen**, una piattaforma social dedicata alla condivisione di ricette tradizionali e alla gestione delle loro varianti ("eredità"). 

Sviluppato con **React**, **Mantine UI** e gestito tramite **Axios** per la comunicazione con le API.

---

### 🔗 Collegamenti del Progetto
* **🚀 Live Demo:** [https://heritage-kitchen-front-end.vercel.app/](https://heritage-kitchen-front-end.vercel.app/)
* **⚙️ Backend Repository:** [github.com/Enxos27/Heritage-Kitchen](https://github.com/Enxos27/Heritage-Kitchen)

---

### ✨ Funzionalità Principali
- **Autenticazione Sicura:** Login e registrazione gestiti tramite JWT (JSON Web Token).
- **Social Feed:** Home page dinamica con caricamento infinito di ricette.
- **Recipe Heritage:** Visualizzazione delle varianti di una ricetta (collegamento padre-figlio).
- **User Experience:** Interfaccia responsive ottimizzata per Mobile (grazie a Mantine AppShell).
- **Search & Explore:** Sistema di ricerca per ricette e suggerimenti di nuovi chef da seguire.

---

### 🛠️ Tech Stack
* **Framework:** React 18
* **Styling:** [Mantine UI](https://mantine.dev/) (Componenti React moderni e accessibili)
- **Icone:** Lucide React
- **Routing:** React Router 6
- **HTTP Client:** Axios (con intercettori per il token JWT)
- **Deployment:** Vercel
